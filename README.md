# katabank
A simple test  Api for a bank account 

## Prerequired
node js should be installed 

## Getting started 

run 
```
npm install
````
````
node app.js
````

1- Open an Api tester like : postman or Swagger

2- test the bank APi 

### Get method
````
localhost:3000/account
````
### Put method

````
localhost:3000/deposit
{ "value" : 100 } 
````
In the body
